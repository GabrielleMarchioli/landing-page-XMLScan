# XMLScan Landing Page

Landing page for XMLScan - Fiscal note (NF-e) validation and reconciliation system.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```
